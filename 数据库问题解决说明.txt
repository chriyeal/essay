# 数据库初始化问题解决方案

## 问题分析
您遇到的错误 `Unknown column 'dept_id' in 'field list'` 是因为：
1. 数据库中可能存在旧版本的 `sys_user` 表，缺少 `dept_id` 字段
2. 直接执行脚本时，CREATE TABLE IF NOT EXISTS 不会修改现有表结构
3. INSERT 语句引用了不存在的字段导致报错

## 解决方案

### 方法一：使用修复版脚本（推荐）
1. 在Navicat中打开 `sql/fixed_init.sql` 文件
2. 执行该脚本（会先删除旧表再重建）

### 方法二：手动清理后再执行
```sql
-- 先删除所有相关表
DROP TABLE IF EXISTS study_statistics;
DROP TABLE IF EXISTS tomato_record;
DROP TABLE IF EXISTS study_plan;
DROP TABLE IF EXISTS sys_user;

-- 然后执行原来的 full_init.sql 脚本
```
