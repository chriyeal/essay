# 个性化学习计划管理系统使用说明

## 🚀 系统启动步骤

### 1. 数据库初始化
使用Navicat或其他MySQL客户端执行以下脚本：
```
d:\essays\RuoYi-Vue-master\sql\full_init.sql
```

这将创建所有必要的表并插入示例数据。

### 2. 启动后端服务
```bash
cd d:\essays\RuoYi-Vue-master\ruoyi-admin
mvn spring-boot:run
```
服务将在 http://localhost:8085 启动

### 3. 启动前端服务
```bash
cd d:\essays\RuoYi-Vue-master\ruoyi-ui
npm run dev
```
服务将在 http://localhost:1117 启动

## 📋 系统功能说明

### 1. 首页
- 展示欢迎信息和个人学习统计数据
- 快捷入口到各个功能模块

### 2. 学习计划管理
**功能特点**：
- ✅ 支持总体计划和今日计划分类管理
- ✅ 计划状态筛选（进行中、已完成、已取消）
- ✅ 统计卡片点击筛选功能
- ✅ 完整的计划信息显示（开始时间、结束时间、难度、进度等）
- ✅ 智能生成学习计划功能

**使用方法**：
- 点击统计卡片可筛选对应状态的计划
- 使用"总体计划"/"今日计划"切换视图
- 新建计划时可设置详细信息

### 3. 番茄钟专注学习
**功能特点**：
- ✅ 智能时间设置功能（根据学科和难度自动推荐时间）
- ✅ 完整的番茄钟计时器
- ✅ 今日记录统计
- ✅ 效率分析图表

**智能设置说明**：
- 数学/物理：30分钟专注 + 5分钟休息
- 英语/语文：20分钟专注 + 5分钟休息  
- 计算机：35分钟专注 + 10分钟休息
- 简单任务：减少10分钟专注时间
- 困难任务：增加10分钟专注时间

### 4. 学习统计
- 学习时长统计
- 番茄钟完成情况
- 学科分布分析
- 学习趋势图表

### 5. 个人中心
- 个人信息管理
- 个性签名设置
- 密码修改
- 学习偏好设置

## 🛠️ 常见问题解决

### 1. 登录页面无法访问
- 确保前端服务在 http://localhost:1117 正常运行
- 清除浏览器缓存和Cookie
- 检查网络连接

### 2. 数据库连接失败
- 确认MySQL服务已启动
- 检查数据库配置：localhost:3306，用户名root，密码123456
- 确保执行了数据库初始化脚本

### 3. 功能异常
- 检查浏览器控制台是否有JavaScript错误
- 确认后端API服务正常运行
- 刷新页面重新登录

## 📝 注意事项
